# PRISMA-style flow reconstruction (2026-06-10)

Retrospective reconstruction of the search → screen → include funnel from the
six raw reports' audit trail. Drafted for Section 4.1 of the paper; the figure
is generated by `analysis/prisma_figure.py` → `analysis/figures/prisma.png`.

**Honest-accounting caveat (stated in the paper, §4.1 and §7):** the search
agents logged queries and per-paper structured entries, but not snippet-level
hit counts. The earliest auditable stage is therefore *structured assessment*,
not the conventional PRISMA "records identified" stage, which cannot be
reported. All counts below are recomputable from the files named.
